#PySpark Housing
Проект по отработке навыков PySpark: предсказание медианной стоимости домов с помощью линейной регрессии.

Описание
В этом проекте используется PySpark для анализа данных о жилье в Калифорнии.
Цель проекта — отработать навыки работы с PySpark, построить модель линейной регрессии и предсказать медианную стоимость домов.
Для оценки модели используются метрики RMSE, MAE и R².
